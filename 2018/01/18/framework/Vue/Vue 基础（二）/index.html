v-show

  • v-if 与 v-show
    • v-show 的元素会始终渲染并保持在 DOM 中，v-if的元素会被移除
    • v-show 不支持 <template> 语法
    • v-show 的元素会始终渲染并保持在 DOM 中，v-if的元素会被移除
    • v-if 是惰性的，只有在条件第一次为 true 的时候才进行局部渲染
    • v-if 有更高的切换消耗，而 v-show 有更高的初始渲染消耗
