Vue 基础（一）

发表于 2018-01-18 | 分类于 framework

Vue 是一套用于构建用户界面的–渐进式框架–。Vue 的核心库只关注视图层，不仅易于上手，还便于与第三方库或既有项目整合。另一方面，当与现代化的工具链以及各种支持类库结合使用时，Vue 也完全能够为复杂的单页应用提供驱动

  • 单文件组件

    • css+html+js 都写在 vue 文件中，使得组件更简洁，清晰
    • angular 中的 js 文件只能写 js
    • react 中可以写 css-in-js，但是缺乏选择器功能

  • 网址

    • https://vuefe.cn/

Vue实例

属性与方法

动态属性

  • 自定义的数据和方法是要分别绑定到绑定到实例的不同属性上
  • 例如数据要绑定要 data 属性，方法要绑定到 methods 方法
  • 实例上的 data 和 methods 里面的 key 值会自动挂载到vue实例上
  • 使用 this 在配置对象内部获取实例上绑定的动态属性

实例属性

  • vue 实例上的 data、methods 等称为实例属性
  • 通过 实例.$实例属性名 获取实例属性
  • 通过 $watch 方法可以监听data属性里面的数据的变化

实例生命周期

注意

  • 和 react 的生命周期基本思想是一样的
    • react 监听 props 和 state 属性的变化
    • vue 监听 data 属性的变化
  • vue中的生命周期函数比react中的方法多
  • vue 作为一个库使用时，不具有生命周期函数

生命周期函数

  • beforeCreate — 创建之前
  • created — 创建之后
  • beforeMount — 挂载之前
  • mounted — 挂载之后，此函数之前的生命周期函数在整个生命周期中只执行一次
  • beforeUpdate — 更新之前
  • updated — 更新之后
  • beforeDestroy — 卸载之前
  • destroyed — 卸载之后

模板语法

Mustache语法

  • Mustache语法内部会按照纯文本输出(双大括号)

  • 在 Mustache 中可以处理一些简单的js表达式

  • Mustache 中可以使用过滤器，也可以多过滤器串联。过滤器可以在用在v-bind中使用 （vue 2.1.0+）

指令

  • v-html — 会按照 HTML 规则去解析内
  • v-once — 执行一次性地插值
  • v-bind — 为标签的属性赋值时不能使用 Mustache语法，需要用v-bind指令
  • v-on — 绑定事件
  • v-for / v-if — 循环或判断
  • 缩写 — v-bind和v-on在模板语法中可以缩写
  • 修饰符 — 指出一个指令应该以特殊方式绑定

计算属性

  • 与Mustache语法相比，计算属性 computed 可以处理模板语法中的复杂业务逻辑，
  • computed 与 methods
    • computed 的依赖如果没有发生变化，再次调用的时候，能够立即返回上次缓存的计算值
    • methods 需要重新执行方法体
  • computed 与 watch
    • watch方法每次只能监听一个 data 值的变化
    • 而计算属性可以同时监听多个 data 值的变化
    • 用计算属性可以简化 watch 中重复的代码
