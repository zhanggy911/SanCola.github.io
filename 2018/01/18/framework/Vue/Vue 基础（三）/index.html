Vue 基础（三）

发表于 2018-01-18 | 分类于 framework

组件概念：

  • 组件可以扩展 HTML 元素，封装可重用的代码
  • 类似于 angualr 中自定义指令，是 vue 中的一种自定义标签
  • 类似于 react 中的通用组件，是高可复用性的（例如：列表，按钮，等待器）

使用组件

全局注册组件

  • 全局组件的定义要在创建根实例之前
  • 在全局注册的组件可以在子组件的页面中任意使用

局部注册组件

  • 在要使用的组价中增加 components 属性，注册引入的组件
  • 一般局部注册的组件都是通过 .vue 文件实现的

Dom模板解析问题

  • 在一些特殊标签如 table ul ol select 中使用自定义组件的时候会有限制
  • 使用来自以下来源之一的字符串模板，这些限制将不适用
    • <script type="text/x-template">
    • JavaScript 内联模版字符串
    • .vue组件

组件组合

  • 父组件通过 prop 给子组件下发数据
  • 子组件通过 event 给父组件发送消息

prop 属性

使用 prop

参数命名

  • 父组件传递参数用 kebab-case（短横线隔开，在 HTML 中）
  • 在子组件接收的时候用 camelCase（驼峰命名，在 JavaScript 中）

动态 prop

  • 如果需要向子组件传递父组件的 data 属性，需要用 v-bind 传递
  • 当父组件的数据变化时，该变化也会传导给子组件
  • 当传递的数据为数字时，v-bind 传递到子组件内部的是数值类型

单项数据流

  • 一般情况下不要在子组件中改变父组件中传递过来的 props，防止子组件影响父组件
  • 需要修改父组件传递过来的 props，需要特别处理
  • 注意：JavaScript 中对象和数组是引用类型，指向同一个内存空间，如果 prop 是一个对象或数组，在子组件内部改变它会影响父组件的状态
