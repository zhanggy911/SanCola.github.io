React Native 基础

发表于 2018-01-18 | 分类于 framework

介绍

React Native (简称RN)是Facebook于2015年4月开源的跨平台移动应用开发框架，是Facebook早先开源的UI框架 React 在原生移动应用平台的衍生产物，目前支持iOS和安卓两大平台。RN使用Javascript语言，类似于HTML的JSX，以及CSS来开发移动应用

  • 网址
    • http://reactnative.cn/
    • http://www.lcode.org/react-native/

快速开始

环境配置

  • Java
  • C++
  • Node
  • Git
  • python
  • Android ADT

    安装 react-native-cli

快速开始

  • 项目初始化
    • react-native init rn_example
  • 手机连接电脑
  • 打包
    • react-native run-android
    • 自动启动一个打包构建服务，端口号是8081
    • 自动将生成的app安装到手机
  • android手机需要打开手机的悬浮框权限
  • 重启应用，通过摇一摇或者菜单键打开开发选项菜单

  • 选择 Dev Settings–>Debug Server host & port for device

  • 设置手机和电脑在同一个局域网的ip地址，添加8081端口

  • 之后就可以进行开发了
  • 修改代码之后，运行手机应用开发菜单中的 Reload 可以刷新应用内容
  • 可以点击手机开发菜单中的 Debug JS Remotely 键，实现在浏览器中断点调试功能

基本使用

  • 只有第一次安装应用的时候需要连接数据线，之后只要在同一个局域网就可以通过手机查看效果
  • 需要启动服务
    • 运行 react-native start ，启动一个服务
    • 在浏览器中输入 http://localhost:8081/index.android.bundle?platform=android
    • 能刷出js代码才算服务启动完毕
    • 该服务在整个开发的过程中不能关闭

RN主要内容

常用组件

在RN应用中不会看到任何的html标签，因为 RN 是原生应用。只能使用官方提供的组件去构建应用，因为只有这些组件才能通过配置文件转化为原生组件

  • View
    • 相当于pc里面的div标签
    • http://reactnative.cn/docs/0.39/view.html#content
  • Text
    • RN中只要涉及到文字，就用这个组件包裹
    • http://reactnative.cn/docs/0.39/text.html#content
  • Image
    • 图片相关内容都需要使用这个组件
    • 使用到图片就要给固定的宽高
    • source属性中不能拼接路径，只能使用拼接好的
    • http://reactnative.cn/docs/0.39/images.html#content
    • http://reactnative.cn/docs/0.39/image.html#content
  • Botton
    • Button TouchableHighlight TouchableNativeFeedback TouchableOpacity TouchableWithoutFeedback
    • http://reactnative.cn/docs/0.39/touchablehighlight.html#content
  • TextInput
    • 涉及到文本输入都要使用该组件
    • http://reactnative.cn/docs/0.39/textinput.html#content
  • ScrollView
    • 滚动的内容都用这个组件包裹
    • http://reactnative.cn/docs/0.39/scrollview.html#content
  • Navigator
    • RN中的路由
    • http://reactnative.cn/docs/0.39/navigator.html#content
  • ListView
    • 列表渲染的东西放在这里面
    • http://reactnative.cn/docs/0.39/listview.html#content

常用api介绍

  • StyleSheet
    • 所有RN中的样式都必须用这个Api创建
    • 所有样式名称都必须是驼峰命名
    • RN中所有的组件默认display属性都是flex，而且主轴方向是column
    • http://reactnative.cn/docs/0.39/stylesheet.html#content
