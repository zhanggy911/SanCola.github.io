PHP 简介

发表于 2018-01-18 | 分类于 PHP

文档定义

  • 结尾必须加 ; 号
  • php 文件内部可以写 html 标签，更可以直接将 html 标签通过修改后缀名称的方式直接转为 php；
  • php 语句只写在要动态生成的地方即可

内容输出

echo

  • PHP 语句直接使用即可，输出字符串

print_r()

  • 函数输出复杂数据类型，比如数组、对象

var_dump()

  • 函数输出详细信息，如对象、数组、数据类型

PHP变量规则

  • 变量以 $ 符号开头，其后是变量的名称
  • 变量名称必须以字母或下划线开头
  • 变量名称不能以数字开头
  • 变量名称只能包含字母数字字符和下划线（A-z、0-9 以及_）
  • 变量名称对大小写敏感
  • 不要使用符号

运算符

  • PHP 中的运算符跟 JavaScript 中的基本一致,用法也基本一致
  • 算数运算符: +、-、/、*、%
  • 赋值运算符: =、+=、-=

数据类型

  • 字符串、整数、浮点数、布尔、数组、对象、NULLL
  • 定义字符串时需要注意:
    • 单引号: ' ' 内部的内容只是作为字符串
    • 双引号: " " 如果内部是PHP的变量,那么会将该变量的值解析
  • 字符串连接
    • PHP使用 . 进行字符串连接

循环语句

  • 只列举 foreach、for循环（其他语句与js相同）

数组

  • 普通数组
  • 关系型数组

函数

  • 与 js 写法基本相同

对象

  • PHP中允许使用对象这种自定义数据类型。使用时必须先声明,实例化之后才能够使用
