发表于 2018-01-18 | 分类于 PHP

表单操作

常用方法

header()

  • 设置编码格式为 utf-8
  • 设置跳转页面
  • 设置页面刷新间隔

$_FILE 文件操作

  • $_FILES — 用法跟 $_GET、 $_POST 类似,都是关系型数组
  • _FILE['key'] — 可以获取对应上传的文件,这里的key跟提交时的name相对应
  • _FILE['key']['name'] — 可以获取上传的文件名
  • _FILE['key']['tmp_name'] — 可以获取上传的文件保存的临时目录

move_uploaded_file()

  • 上传的临时文件,一会就会被自动删除,我们需要将其移动到保存的位置
  • 参数:
    • 参数1: 移动的文件
    • 参数2: 目标路径

文件读写

  • file_get_contents('url')
  • file_put_contents('url')

数组方法

  • in_array（查找内容，查找的数组）— 判断数组中是否有相应值；
  • count($数组名) — 获取数组的length；
  • array_rand(数组名称，返回数量) — 随机返回n个数组中的数值(实际返回的是索引值，需要进一步操作)；
  • array_key_exists(key, 数组) — 非空判断
