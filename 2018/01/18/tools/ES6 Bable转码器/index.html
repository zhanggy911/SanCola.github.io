Bable转码器

发表于 2018-01-18 | 分类于 tools

Babel 是一个广泛使用的 ES6 转码器，可以将 ES6 代码转为 ES5 代码，从而在现有环境执行。

.babelrc配置文件

  • 存放在项目根目录下，只能使用编辑器之类的工具创建
  • 填写好该配置文件后才能使用 Babel
  • presets 字段 — 用于填写转码规则
  • babel的工具都需要使用。
  • 将字段加入 .babelrc 的 presets 字段

babel-cli转码工具

  • babel-cli 是 Babel 自带的命令行转码工具
  • 基本命令

babel-register

  • 使用 babel-register 作为开发的编译转换环境，可以在代码运行的过程中实时编译转换
  • 单独安装
  • 需要新建一个最顶层的入口文件，并加载 babel-register
  • 只对 require 加载的文件转码，不会转换当前文件
  • 实时转码，不适合用于生产环境

其他 Babel 插件

  • babel-polyfill
    • Babel 默认只转换新的 JavaScript 句法（syntax），而不转换新的 API，举例来说，ES6在 Array 对象上新增了 Array.from 方法，Babel 并不会转码这个方法。需要使用 babel-polyfill ，为当前环境提供一个垫片才能转换
  • babel-core
    • 提供 Babel 的 API 以供某些代码进行转换

  • babel-node

    • Bebel 自带命令，提供支持 ES6的 REPL 环境，可以直接运行 ES6 的代码
  • Babel 项目安装时，可以使用 babel-node 替代 node ，替代后，script.js 不再需要转码处理，但是需要改写 package.json 文件
