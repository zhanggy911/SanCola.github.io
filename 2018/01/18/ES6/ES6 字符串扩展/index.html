ES6 字符串扩展

发表于 2018-01-18 | 分类于 JavaScript

includes()、startsWith()、endsWith

  • include()
    • 返回布尔值，表示是否找到了参数字符串
  • startsWith()
    • 返回布尔值，表示参数字符串是否在源字符串的头部
  • endsWith()
    • 返回布尔值，表示参数字符串是否在源字符串
  • 上面三个方法都支持第二个参数，表示从何处开始搜索

repeat()

  • 返回一个新的字符串，用于将源字符串重复 n 次

padStart()、padEnd()

  • padStart()
    • 头部补全
  • padEnd()
    • 尾部补全
  • 接收两个参数，分别用来指定字符串最小长度，以及指定用来补全的字符串

模板字符串

  • 使用 ` ` 标识字符串
  • 嵌入字符串需要将变量名写在 ${ } 中
  • 模板字符串表示多行字符串时，所有空格和缩进会保留
  • 大括号内部可以放入任意 JavaScript 表达式，也可以调用函数
  • 模板字符串支持嵌套
  • 允许引用模板字符串，在需要的时候执行
