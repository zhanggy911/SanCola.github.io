ES6 函数扩展

发表于 2018-01-18 | 分类于 JavaScript

函数参数的默认值

  • ES6允许为函数的参数指定默认值
  • 函数参数是默认声明的，函数内部不能使用 let , const 声明函数参数的同名变量
  • 指定默认值时，函数不能有同名参数
  • 参数默认值是惰性求值的
  • 结合解构赋值使用
  • 参数默认值的位置
    • 定义了默认值的参数，应当是函数的尾参数
    • 无法省略默认值后面的参数
    • 显示输入 undefined 将触发参数等于默认值
  • 函数的 length 属性
    • 指定函数参数默认值后，函数 length 属性指的是不包含默认值的参数个数
  • 作用域
    • 一旦设置了参数的默认值，函数进行声明初始化时，参数会形成一个单独的作用域（context）
    • 初始化结束，这个作用域就会消失

rest 参数

ES6 引入 rest 参数（形式为“…变量名”），用于获取函数的多余参数，这样就不需要使用arguments对象了。rest 参数搭配的变量是一个数组，该变量将多余的参数放入数组中

  • rest 只能是最后一个参数
  • 函数的 length 属性不包含 rest 参数

扩展运算符

扩展运算符（spread）是三个点（…）用于将一个数组转为用逗号分隔的参数序列

  • 用于替代数组的 apply 方法

箭头函数

  • 允许使用 => 定义函数
  • 如果箭头函数不需要参数或需要多个参数，就使用圆括号代表参数部分

  • 如果箭头函数的代码多余一条语句，需要使用大括号并使用 return 返回

  • 结合解构赋值使用

  • 结合 rest 参数使用
  • 使用注意
    • 箭头函数内部 this指向定义时的对象而不是使用时的对象，箭头函数不具备 this
    • 不能作为构造函数
    • 不能使用 arguments 对象，可以使用 rest 参数替代
    • 不能使用 yield 命令，不能作为 Generator 函数
