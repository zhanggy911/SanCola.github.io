jQuery DOM操作

发表于 2018-01-18 | 分类于 library

样式操作

获取样式

  • 语法：$(selector).css('class');

设置样式

  • 设置单个样式：
    • 第一个参数表示：样式属性名称
    • 第二个参数表示：样式属性值
    • $(selector).css('color', 'red');
  • 设置多个样式：
    • 参数为对象
    • $(selector).css({'color':'red', 'font-size':'30px'});

尺寸位置

尺寸操作

  • height() 与 width() — 设置或者返回元素的高度及高度,返回结果是数值类型。
  • innerWidth() 与 innerHeight() — 包括 padding
  • outerWidth() 与 outerHeigth() — 包括 padding、border
  • outWidth(true) 与 outerHeight(true) — 包括 padding、border、margin
  • 只有 height() 与 width() 可以进行设置操作，innerWidth()、outWidth() 都是只读属性

位置操作

  • scrollTop() — 获取或者设置元素垂直方向滚动的位置
  • scrollLeft() — 获取或者设置元素水平方向滚动的位置
  • offset() — 获取或设置元素相对于document左上角的位置
  • position() — 获取相对于其最近的有定位的父元素的位置。相当于之前JS中的offsetLeft和offsetTop

类名操作

添加类名

  • 语法：addClass(className)
  • 注意
    • 把名字传入即可，不要加点

移除类名

  • 语法：removeClass(className)
  • $(selector).removeClass('liItem') --- 移除指定类名
  • $(selector).removeClass() --- 不指定参数，表示移除所有类名

类名判断

  • 语法：hasClass(calssName)
  • $(selector).hasClass('liItem') --- 返回true或false

切换类名

  • 语法：toggleClass(className)
  • 切换类名，如果没有指定类名就添加，有就移除。并不会影响原来的类名
  • $(selector).toggleClass('liItem');

节点操作

添加元素

  • append(htmlString / jQuery对象)
    • 在被选元素内部的最后一个子元素（或内容）后面插入内容（页面中存在或者创建出来的元素都可以）
    • 是页面中存在的元素，那么调用 append() 后，会把这个元素从原先的位置移除，然后再插入到新的位置
    • 给多个目标追加一个元素，append() 方法内部会将这个元素复制多份，然后追加到多个目标中
  • prepend()
    • 作用：在元素的第一个子元素前面追加内容或节点
    • $(selector).prepend(node);
  • after()
    • 作用：在被选元素之后，作为兄弟元素插入内容或节点
    • $(selector).after(node);
  • before()
    • 作用：在被选元素之前，作为兄弟元素插入内容或节点
    • $(selector).before(node);

创建元素

  • html(htmlString)

清空元素

  • empty(htmlString)
  • remove(htmlString)
  • html(htmlString)

复制元素

  • $(selector).clone();
  • 传入参数ture会将改元素的事件一起复制

表单操作

获取 / 设置属性

  • attr()

移除属性

  • 参数为：要移除的属性的名称
  • $(selector).removeAttr(“title”);
  • 注意
    • 表单元素中属性值为布尔值的属性，如checked、selected、disabled要使用.prop()方法

代码/文本/值 操作

  • html()

    • 作用：设置或获取匹配元素的代码内容

  • text()

    • 作用：设置或获取匹配元素的文本内容

  • val()

    • 作用：设置或返回表单元素的值，例如：input,select,textarea的值

异步表单文件提交

  • new FormData()

实时监听表单输入事件

  • input propertychange

动画函数

隐藏显示动画

  • show()
  • hide()
  • toggle()

滑入滑出动画

  • sildeDown()
  • slideUp()
  • slideToggle()

淡入淡出动画

  • fadeIn()
  • fadeOut()
  • fadeToggle()
  • fadeTo()

自定义动画

  • animate(prop属性的目标值，speed速度，easing动画执行的方式，回调函数)
    • prop可以用对象的方式传入多个参数，实现同时调整；
    • 默认即缓动动画，swing（缓动），linear（匀速）
  • 所有能够执行动画的属性必须只有一个数字类型的值。

停止动画

  • $(selector).stop(clearQueue,jumpToEnd)
    • 第一个参数表示是否清空所有的后续动画
    • 第二个参数表示是否立即执行完当前正在执行的动画
  • 作用：停止当前正在执行的动画
    • 如果多个动画在同一元素上执行，对这个元素来说，后面的动画将被放到队列中，从而形成动画队列-
  • 解释
    • 当调用stop()方法后，当前动画停止于当前样式的值（不会到达目标值），队列中的下一个动画会立即开始
    • 如果参数clearQueue被设置为true，那么队列中剩余的动画就被删除了，永远不会再执行。
    • 如果参数jumpToEnd被设置为true，那么当前动画会停止，但是参与动画的每一个CSS属性将被立即设置为它们的目标值。比如：slideUp()方法，那么元素会立即隐藏掉。如果存在回调函数，回调函数也会立即执行。
  • 注意
    • 如果元素动画还没有执行完，此时调用sotp()方法，那么当前动画将会停止。并且由于本次动画没有执行完成，本次动画的回调函数也不会被执行（下一次的会照常执行）。
  • 常用方式
    • $(selector).stop() — 当前动画停止于此刻的样式状态 后面的动画继续执行
