Nunjucks 模板引擎

发表于 2018-01-18 | 分类于 NodeJS

Nunjucks是Mozilla开发的一个纯JavaScript编写的模板引擎，主要运行在Node环境下。Nunjucks就是用JavaScript重新实现了jinjia2。

安装

  • 项目安装
  • npm install nunjucks --save

配置

  • 引入项目依赖包
  • const nunjucks = require('nunjucks')

注意

  • nunjucks 模板引擎没有对模板文件名的后缀名做特定限制
    • 如果文件名是 a.html 则渲染的时候就需要传递 a.html
    • 如果文件名是 b.nujs 则传递 b.nujs
  • Nunjucks默认使用同步IO读取模板文件
    • Nunjucks会缓存已读取的文件内容，后续请求不会再次读取文件的
    • 缓存开启需要指定noCache: false参数
    • 开发环境下，可以关闭cache，便于实时修改模板
    • 生产环境下，需要打开cache，避免出现性能问题
