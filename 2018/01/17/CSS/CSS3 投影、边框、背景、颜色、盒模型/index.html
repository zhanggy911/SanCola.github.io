CSS3 投影、边框、背景、颜色、盒模型

发表于 2018-01-17 | 分类于 CSS

投影

  • box-shadow — 设置投影样式
    • h-shadow — 必需。水平阴影的位置。允许负值
    • v-shadow — 必需。垂直阴影的位置。允许负值。
    • blur — 可选。模糊距离。
    • spread — 可选。阴影的尺寸(扩展)
    • color — 可选。阴影的颜色。请参阅 CSS 颜色值。
    • inset — 可选。将外部阴影 (outset) 改为内部阴影。
    • 可使用 ， 号分割来添加多个阴影
  • text-shadow — 设置文本投影样式，用法与 box-shadow 相同

边框

  • border-radius — 圆角
    • 1个值： 上下左右
    • 2个值：左上／右下，右上／左下
    • 3个值：左上，右上／左下，右下
    • 4个值：左上，右上，右下，左下
    • 允许单独设置
  • border-collapse: separate — 表格圆角
  • border-image — 图片边框
    • source — 图片路径
    • slice — 尺寸分割
    • repeat: repeat — 默认分割方法
    • repeat: round — 中间线段拉伸

背景

渐变

注意：渐变属性生成的是一张图片，可能存在加载不全的问题

  • 线性渐变（linear-gradient）
  • 径向渐变（radial-gradient）

背景图片

  • background-size — 设置背景图片缩放大小
    • cover — 背景图片原比例缩放到完全填满 可能会超出
    • contain — 背景图片原比例缩放到 高度 或者宽度 跟容器一样 可能会有空白
  • background-repeat — 设置背景图片缩放
    • round — 自动缩放 直到填满
    • space — 两个方法都缩放,直到某一边充满容器为止
  • background-clip — 设置背景图片的开始位置
    • border-box — 从border开始 虚线 才能看到
    • padding-box — 从 padding开始的位置
    • content-box — 从内容的位置开始
  • background-origin — 设置背景图片的 裁切位置
    • padding-box — 从padding部分 开始裁切
    • content-box — 从内容区域开始裁切
  • 多重背景

颜色

  • RGBA模式
    • Red — 0-255
    • Green — 0-255
    • Blue — 0-255
    • Alpha — 0-1（小数）
    • 注意，不使用A那么认为没有透明度
  • HSLA模式
    • Hue(色调) — 0-360
    • Saturation(饱和度) — 0-100%
    • Lightness(亮度) — 0-100%
    • Alpha(透明度) — 0-1（小数）

透明度说明：

  • opacity 只能针对整个盒子设置透明度，子盒子及内容会继承父盒子的透明度
  • rgba 不会继承透明度；
  • transparent 始终完全透明，他是作为颜色的取值来使用的color:transparent

盒模型

  • box-sizing — 改变盒模型对元素宽高的计算方式
    • content-box — 系统默认
    • border-box — 系统计算实际宽高不在与padding／border相关，即padding／border全部包含在设置的width／height内部
