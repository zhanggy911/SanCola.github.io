CSS 元素特性、分类与转换、部分常用样式

发表于 2018-01-17 | 分类于 CSS

css 元素特性

  • CSS选择器权重
    • 标签选择器 < 类选择器 < id选择器 < 行内样式 < important
    • 选择器复用可以使权重叠加
  • 层叠性
    • 同一标签，权重相同的情况下定义的样式发生冲突，最后定义的样式会将前面定义的样式覆盖掉
  • 继承性
    • 继承性只能在嵌套关系的元素之间产生，子元素没有设置样式，但是会受到父元素的样式影响

元素分类

块级元素（block）

  • 元素
    • div 、p、ul ….
  • 特点
    • 如果块级元素没有设置宽高，那么它的默认宽度是其父元素的宽度
    • 宽度高度不能继承
    • 块级元素可以直接设置宽度和高度，独占一行显示

行内元素（inline）

  • 元素
    • span、a、font、b、s、strong …
  • 特点：
    • 行内元素不能直接设置宽高
    • 行内元素会直接在同一行内显示

行内块元素（inline-block）

  • 元素
    • img、input …
  • 特点：
    • 多个行内块元素可以在同一行上显示
    • 可以直接设置宽度和高毒

元素之间的转换

  • 转换为块元素
    • display: block
      • 转换为行内块元素
      • display: inline-block
      • 转换为行内元素
      • display: inline

常用样式

伪类

以 <a> 标签为例：

  • a:link{属性：值 } — 设置a标签的默认样式，推荐还是使用a{属性：值}进行设置
  • a:visited{属性：值} — 设置a标签访问过后的样式（只能设置颜色相关的属性）
  • a:hover{属性：值} — 设置鼠标移动到超链接上的样式
  • a:active{属性：值} — 设置点击瞬间的样式
  • input:focus{属性} — 设置获取焦点时的样式

标签的伪类可以连续设置，但是需要按照顺序：L(link) V(visited) H(hover) A(active)

背景

  • background-image: url('图片路径') — 设置图片背景
  • background-repeat — 设置背景图片的平铺方式
    • no-repeat — 不平铺
    • Repeat-x — 横向平铺
    • Repeat-y — 纵向平铺
  • background－position — 设置背景图片位置
    • 可以设置 left/right 等方位名词，也可以设置数字
    • 只要有一个具体数字，且不管该数字在哪个位置出现，那么第一个参数就代表水平方向
    • 如果是方位名词，则没有先后顺序，只有一个值的时候，默认值为center
  • background属性连写：没有个数及先后顺序的限制，但是必须有大小

文字

  • font-style — 字体风格
    • italic — 斜体
    • oblique — 图形化斜体
    • normal — 默认值
  • font-weight — 字体粗细，一般设置400为normal，700为bold
  • font-size — 字体大小
  • line-height — 字体行高，可以与font-size连写
  • font-family — 指定字族
  • font属性连写: 有先后顺序，不用一次指定所有值。
  • text-align — 内容左右居中（设置在父元素上）
  • text-indent — 文字首行缩进
  • white-space: nowrap — 强制文本不换行
  • word-wrap:break-word — 自动换行
  • word-break: normal — 自动换行
  • word-break: break-all — 强制英文单词换行
  • text-overflow：ellipsis — 设置文字溢出时使用省略号

颜色

  • color: #999 — 16进制表示
  • color:rgb(12,45,165) — rgb表示
  • color:rgba(0,0,0,.5) — rgba表示，最后一位使用0-1表示透明度，不会使子元素继承透明度
  • opacity: 0.5 — 表示透明度，会连带子元素继承透明度属性

鼠标样式

  • cursor — 鼠标样式（chrome）
    • cursor: not-allowed — 禁止点击
    • …
