CSS 对齐、隐藏、布局方式等

发表于 2018-01-17 | 分类于 CSS

对齐

绝对定位的元素居中显示

  • 子元素设置
    • left: 50%
    • margin-left: -(自身宽度的一半)
    • 上下居中同理

行内块元素对齐方式

  • vertical-align — 对齐方式
    • baseline — 默认基线对齐
    • top — 顶部对齐
    • middle — 中间对齐
    • bottom — 底部对齐
  • 设置图片垂直居中
    • 设置行高使文字居中
    • 设置图片：vertical-align: middle

文字对齐

  • text-align
    • right
    • left
    • center
    • justify — 两端对齐

文字两端对齐

  • text-align: justify
  • text-justify: inter-ideogra

块级元素快速居中

  • margin: 0 auto
    • 使元素水平居中，此种方法父元素、子元素必须设置宽度
  • margin: auto 0
    • 元素垂直居中，同理父元素，子元素必须设置高度

隐藏

  • overflow — 设置超出父元素边界的部分的显示方式
    • visible — 默认，超出部分显示
    • hidden — 溢出父元素部分隐藏
    • scoll — 添加固定样式的滚动条，无论元素是否超出
    • auto — 自适应添加滚动条，常用
  • display — 设置元素本身是否显示
    • block — 显示元素
    • none — 隐藏元素， 元素不占位置
  • visibility-hidden — 隐藏元素，元素占位置
  • visibility-collapse — 隐藏表格内的一行或一列，但是不影响表格布局

常见网页布局方式

固定布局 fixed Layout
响应式布局 Responsive Layout
流式布局 Fluid Layout
栅格布局 Grid Layout
瀑布流布局 Waterfall Layout
圣杯布局 Holy Grail Layout

常用技巧

logo内容移除

  • 页面内logo要将logo图片放置在a标签内
  • logo要以背景图片的方式出现 （< image src=”地址”>）
  • 移除logo中的文字内容，设置 text-indent 为负值

雪碧图使用

  • 由多张小图标、按钮等图片组成的一张不带背景的大图（减少网页请求）
  • 使用背景图片定位的方式来显示想要的部分

滑动门

  • 含义是导航栏、标签页等元素宽度可以根据内容缩放
  • 使用浮动将元素一行显示，分左中右等多个部分设置相应的背景图片
  • 设置内部文字／元素的padding值撑起宽度，达到根据元素宽度变化来设置标签页样式的效果
