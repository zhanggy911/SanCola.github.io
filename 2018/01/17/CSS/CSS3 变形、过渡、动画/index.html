CSS3 变形、过渡、动画

发表于 2018-01-17 | 分类于 CSS

变形

使用变形不会对其他元素产生影响，所有附属属性都可以写在一行内，用空格隔开

2D 变形

  • transfrom-translate — 移动
    • 单独设置
      • transfrom-translateX()
      • transfrom-translateY()
      • transfrom-translateZ()
    • 同时设置
      • transfrom-translate（x,y,z）
  • transfrom-scale — 缩放
    • 单独设置
      • transfrom-scaleX9()
      • transfrom-scaleY()
      • transfrom-scaleZ()
    • 同时设置
      • transfrom-scale（x,y,z）
  • transfrom-rotate — 旋转
    • 单独设置
      • transfrom-rotateX()
      • transfrom-rotateY()
      • transfrom-rotateZ()
    • 同时设置
      • transform-rotate（x,y,z）
  • transfrom-skew — 变形
    • 单独设置
      • transfrom-skewX()
      • transfrom-skewY()
      • transfrom-skewZ()
    • 同时设置
      • transfrom-skew（x,y）

3D 属性

  • perspective
  • perspective-origin
  • preserve-3d
  • backface-visibility

transfrom-origin

  • 系统默认元素中心缩放，所以需要使用origin修改
  • 一般将重心点设置在变化之前的样式里
  • 可以使用文字方向，也可以使用直接坐标设置

过渡

  • transition — 过渡属性
    • property — 需要使用过渡的属性名称
    • duration — 持续时间
    • timing-function — 动画方式
    • delay — 延迟时间
  • 属性连写
  • 多属性合写
  • 贝塞尔曲线动画
  • step动画

动画

定义动画

  • 支持分段动画，50%{ } 60%{ } 执行完成之后动画会自动回到原点的位置

设定动画执行

  • 一行合写的方式设置动画需要注意：第一次出现的是持续时间，第二次出现的是间歇时间，其他随意
