CSS3 选择器

发表于 2018-01-17 | 分类于 CSS

符号选择器

  • E[att]：包含attr属性
  • E[att="val"]：属性值为val（必须为val）
  • E[att~="val"]：属性值使用空格进行分割,有一个为val（必须以空格分割）
  • E[att^="val"]：属性值以val开头
  • E[att$="val"]：属性值以val结尾
  • E[att*="val"]：属性中包含val
  • E[att|="val"]：属性以‘-’分割，其中有val值（如果属性只有val 那么也会被选中哦）

E表示标签名，att表示属性名

兄弟选择器

  • 相同父元素中，选中某个元素之后的所有符合要求的兄弟元素

伪类选择器

  • 结构伪类
    • E:first-child：第一个子元素
    • E:last-child：最后一个子元素
    • E:nth-child(n)： 第n个子元素，计算取值范围是E元素的全部兄弟元素
    • E:nth-last-child(n)： 跟 E:nth-child(n) 类似 ，只是倒着计算
  • 以下是不考虑兄弟元素，只考虑本身的标签索引的写法
    • E:first-of-type
    • E:last-of-type
    • E:nth-of-type
    • E:empty：指的是E标签，并且内容为空
    • E:not(选择器)：选中不满足括号内选择器条件的所有元素E
  • 关于n的取值范围：
    • 当n做为一个独立值时，n取值为n>=1，例如 nth-child(n)
    • 当n做一个系数时，n取值为n>=0者n<0，例如 nth-child(2n+1)、nth-child(-1n+5)；此处需要理解2n+1或者-n+5做为一个整体不能小于1；

伪元素选择器

  • before&after 选择器(主要用来添加元素)
    • 需要配合 content 属性使用（不添加就看不到元素）
    • content 内部可以写文本，但是不能写样式标签
    • 添加的伪元素是行内样式，想要显示必须 display：block
    • 单标签不支持伪元素
    • 只能添加befor／after两个伪元素
  • E:before：在元素E之前添加
  • E:after：在元素E末尾添加
  • E:first-letter
    • 获取的是内容的第一个文字
    • 根据语言不同，会有细微差别
  • E:first-line
    • 获取的是内容的第一行的内容（自适应）
    • 可以配合p标签使用
  • E::selection
    • 选中时的样式
    • 只能设置 background-color、color、text-shadow等属性
  • E::placeholder
    • 该属性处于测试阶段，需要配合浏览器前缀使用
    • 设置 input 标签 placeholder 属性的显示颜色
