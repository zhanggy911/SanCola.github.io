HTML 基本介绍、标签

发表于 2018-01-17 | 分类于 HTML

HTML简介

html，即超级文本标记语言 是标准通用标记语言下的一个分支，是一种符合标准的文档形式

通过 标签 标记要显示的网页中的各个部分。网页文件本身是一种文本文件，通过在文本文件中添加标记符，可以告诉浏览器如何显示其中的内容（如：文字如何处理，画面如何安排，图片如何显示等）。浏览器按顺序阅读网页文件，然后根据标记符解释和显示其标记的内容，对书写出错的标记将不指出其错误，且不停止其解释执行过程，需要注意的是，对于不同的浏览器，可能会有不同的显示效果。

  • HTML 指的是超文本标记语言 (Hyper Text Markup Language)
  • HTML 不是一种编程语言，而是一种标记语言 (markup language)
  • 标记语言是一套标记标签 (markup tag)
  • HTML 使用标记标签来描述网页

Doctype写法及对应html版本

<!DOCTYPE> 不是 HTML 标签。它为浏览器提供一项信息（声明），即 HTML 是用什么版本编写的。只有完全明白页面中使用的确切 HTML 版本，浏览器才能完全正确地显示出 HTML 页面。

html4的三种写法

  • HTML 4.01 Strict Emmet语法为: html:4s tab
  • HTML 4.01 Transitional Emmet语法为: html:4t tab
  • HTML 4.01 Frameset

html5写法

html5+tab 或 ！+tab

emmet文档结构

  • html:5或!：HTML5文档类型
  • html:xt：XHTML过渡型文档类型
  • html:xs：XHTML严格型文档类型
  • html:4t：HTML4过渡型文档类型
  • html:4s：HTML4严格型文档类型

HTML标签

HTML 标记标签通常被称为 HTML 标签 (HTML tag)

  • HTML 标签是由尖括号包围的关键词，比如 <html>
  • HTML 标签通常是成对出现的，比如 <b> 和 </b>
  • 标签对中的第一个标签是开始标签，第二个标签是结束标签
  • 开始和结束标签也被称为开放标签和闭合标签

HTML标签属性

  • HTML 标签可以拥有属性。属性提供了有关 HTML 元素的更多的信息。
  • 属性总是以名称/值对的形式出现，比如：name="value"
  • 属性总是在 HTML 元素的开始标签中规定。

head标签

<head> 标签用于定义文档的头部，它是所有头部元素的容器。

<head> 文档的头部描述了文档的各种属性和信息，包括文档的标题以及和其他文档的关系等。实际上 <head> 标签并不会包含任何用户可见的内容

head内部常用标签

meta标签

  • <mata charset="utf-8"> — 指定文档使用的字符集
  • <meta name="keywords" content="关键字"> — 指定关键字
  • <meta name="discription" content="描述描述"> — 定义搜索引擎描述文本
  • <meta http-equiv="Refresh" content="2；URL=http://www.net.cn/"> — 定义网页重定向

Base标签

  • <base target="_blank"> — 新建页面跳转
  • <base target="_self"> — 在网页内跳转，默认值

Link标签

  • <link rel="stylesheet" href="1.css"> — 引用外部css样式表
  • <link rel="icon" href="favicon.ico"> — 指定网页标签图标”
  • <link rel="shortcut icon" href="favicon.ico"> — 指定网页标签图标”

常用属性

  • hreflang — 被链接文档中文本的语言
  • media — 被链接文档将被显示在什么设备上
  • rel — 被链接文档与当前文档之间的关系（常用的有stylesheet和shortcut icon）
  • sizes — 网站图标的尺寸（仅适用于 rel=”icon”）
  • target — 被链接文档在哪个窗口或框架中加载
  • type — 被链接文档类型

body标签

body 元素是定义文档的主体。

body 元素包含文档的所有内容（比如文本、超链接、图像、表格和列表等等。）body是用在网页中的一种HTML标签，标签是用在网页中的一种HTML标签，表示网页的主体部分，也就是用户可以看到的内容，可以包含文本、图片、音频、视频等各种内容！

body内部常用标签

基本标签

  • 注释标签：<!--><-->
  • 换行标签：<br>
  • 横线标签：<hr>
  • 分段标签：<p>
  • 标题标签：<h1></h1>
  • 加粗标签：<strong></strong> — <b></b>
  • 斜体标签：<em></em> — <i></i>
  • 删除线标签：<del></del> — <s></s>
  • 超链接标签：<a></a>
  • 区域标签：<div></div>
  • 行内区域标签：<span></span>
  • 图片引用标签：<img><img>

基本标签常用属性

  • href — 用于建立链接关系，常用<link>，<a> 标签，解析到此处时浏览器不会暂停解析
  • src — 用于置换元素，且被置换的元素不受css样式限制，浏览器会暂停解析，如<img>，<script>，<video>等
  • alt — 常用于用于图片不能加载时的显示文字
  • title — 当鼠标悬停在该标签时出现的提示信息

注意

  • 为了做到结构样式分离，i，s，em这几种表示样式的标签已经废弃了
  • 由于i标签正好是icon图标这个英文单词的首字母，并且只有一个字母，于是bootstrap将其用于表示图标，后来渐渐传开，再后来s 和 em 这些也用来表示图标了。
