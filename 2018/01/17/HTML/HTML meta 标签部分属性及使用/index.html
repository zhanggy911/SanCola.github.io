HTML meta 标签部分属性及使用

发表于 2018-01-17 | 分类于 HTML

所有的浏览器都支持meta标签，用于提供页面相关信息，信息都是以名（http-equiv和name标示）/值（content标示）对的形式现实。

http-equiv 属性

http-equiv属性会在HTTP头部制定一些服务器返回给浏览器的有用信息

  • content-type — 设置页面的字符集
  • expires — 设置页面的过期时间
  • refresh — 自动刷新到新页面
  • set-cookie — 如果cookie过期，则自动删除本地cookie
  • prama — 禁止浏览器从本地调用缓存
  • window-target — 强制页面在浏览器窗口独立显示
  • page-enter/page-exit — 设定进入页面和退出页面的效果
  • cache-control

name 属性

name属性把content内容关联到一个名称上，html和xhtml都没有固定的name值，可以自由使用

  • author — 制定页面的所有者／作者
  • description — 页面描述，用于搜索引擎收录
  • keywords — 页面描述，用于被搜索引擎收录
  • generator — 用于说明网站制作生成工具
  • revised — 网页文档的修改时间
  • viewport — 用于控制移动端控制页面的缩放／尺寸
  • copyright — 网页版权信息
  • renderer — 使用哪种模式显示网页
