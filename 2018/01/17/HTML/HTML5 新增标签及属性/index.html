HTML5 新增标签及属性

发表于 2018-01-17 | 分类于 HTML

部分新语义标签

这些标签主要用于划分页面结构，语义性更强，但是存在兼容问题。可以使用兼容框架：html5shiv.js 并配合条件注释解决兼容问题

  • nav — 导航
  • header — 头部
  • footer — 页脚
  • main — 文档主要内容
  • artical — 文章
  • aside — 主题内容之外

条件注释语句

emmet: cc:ie6 + tab

多媒体标签

audio 标签

  • src — 地址
  • controls — 控制器
  • autoplay — 自动播放
  • loop — 循环

注意:

  • 由于视频格式问题，不同的浏览器对于相同格式的视频支持不同，需要准备多份视频
  • 无法对播放的视频提供很好的保护效果,因为用户可以直接对视频文件另存为

兼容写法:

video 标签

  • 用法与audio标签基本相同，只是可以设置宽度和高度了，但是视频的宽高比是不变的
  • 进度条标签
    • process
    • meter
    • 存在兼容问题较大

新表单控件及属性

表单控件

  • 新的表单控件在PC端的浏览器样式差距很大
  • 在移动端效果不错，不过使用时仍需要检查

新表单属性

  • placeholder — 默认值点击消失
  • autofocus — 自动获取焦点，一个页面只建议使用一次
  • multiple — 在提交文件时可以选中多个文件
  • autocomplete — 自动完成表单；但是必须具备name值，且表单必须提交过一次
  • require — 验证用户是否输入
    • pattern自定义验证的规则 正则表达式
