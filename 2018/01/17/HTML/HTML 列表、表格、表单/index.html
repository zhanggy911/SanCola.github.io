常规写法

通常意义上，<thead></thead>，<tbody></tbody>等标签是可以不写的

表格常用属性

  • cellpadding — 单元格内容与单元格左边的距离
  • cellspacing — 默认为2，单元格之间间距
  • align — 内容对其方式，常用值：left / right / center
  • colspan — 合并横向单元格 <td colspan=”2”></td>
  • rowspan — 合并纵向单元格 <td rowspan=”2”></td>

表单

一套完整的表单

表单常用标签

  • <form></form> — 表单域
  • <fieldset></fieldset> — 表单分组
  • <legend></legend> — 表单标题
  • <input> — 表单控件

表单常用属性

  • action — 指定信息提交到相应处理程序
  • method — 信息提交方式
  • method = ‘get’ — get方式提交
  • method = ‘post’ — post方式提交

get方式提交的表单信息，会将表单控件里的具有name属性的值，以查询字符串的方式，提交到action内的地址，并且可以在浏览器地址栏内看到

post是将整个表单以请求体的方式发送出去

表单控件及常用属性

  • 点击选中
  • 输入框
  • 密码框
  • 单选按钮
  • 多选按钮
  • 普通按钮
  • 图片提交按钮
  • 上传按钮
  • 提交按钮
  • 隐藏控件
  • 重置按钮
  • 多行文本域
  • 下拉列表

表单控件常用属性

  • input type — 表单类型
  • name — 为该表单命名
  • maxlength — 最大输入长度
  • value — 默认显示文本
  • readonly — 只读
  • disabled — 禁用
  • Radio — 单选控件
  • checked — 设置默认选中项，如果需要radio（单选控件）起作用，必须设置两个单选框的 name 相同
  • Multiple=”meltiple”（允许多选属性）
