JavaScript 对象、创建对象

发表于 2018-01-17 | 分类于 JavaScript

对象

对象的概念

  • 对象是对客观事物的抽象，事物的特征可以用属性表示，事物的行为可以用方法表示
  • JavaScript中，对象就是无序的键值对的集合
  • 对象就是一种数据类型，通过对象可以方便地对变量和函数进行管理
    • 对象具有属性和方法
  • 属性：用来描述对象的特征 一般是名词 对应变量
  • 方法：用来描述对象的行为 一般是动词 对应函数

对象键值对

  • 键值对就是一种对应关系，通过键能够方便地找到值
  • 键:值 key:value、k:v

访问对象的方式

  • 点语法
  • 中括号

对象的动态特性

  • 对象被创建出来后，随时可以修改，这被称为对象的动态特性
  • Object.defineProperty 可以修改属性

对象的遍历

  • 使用 for…in 语句对对象进行遍历

批量创建对象

工厂模式

  • 将创建对象的过程封装进一个函数，只把发生变化的属性作为参数传入
  • 工厂模式只是创建出来一个普通的对象并将其返回，因此无法判断实例的具体类型

自定义构造函数

  • 利用 new 关键字可以创建新的对象
  • new 关键字可以让构造函数中 this 的指向改变，并让构造函数把 this 返回。
  • 构造函数也是函数，只不过会默认返回一个对象
  • 通过构造函数创建对象更方便（不需要创建对象并返回）。
  • 可以通过 instanceof 来判断创建出来的实例是否属于某个对象
  • 工厂模式与自定义构造函数的区别
    • 工厂模式和自定义构造函数（寄生模式）都可以用来创建对象
    • 工厂模式是函数调用模式，寄生模式是构造函数调用模式，this的指向并不相同

修改对象

  • Object.defineProperty
    • 用来给对象的属性设置访问器、能否修改、遍历等
    • 语法
      • Object.defineProperty(obj, propertyName, descriptor)
      • obj 就是要给谁添加属性
      • propertyName 要添加的属性名
      • descriptor 属性的描述信息，也就是特性集合，是一个对象
  • descriptor 对象中的属性
    • writable 默认为false，表示属性是否可写
    • configurable 默认为 false， 表示属性是否可以配置（删除）
    • enumerable 默认为false， 表示属性是否可以被遍历（for-in）
    • value 用来设置属性的值
    • get 和 set 方法，和 writable 一样都可以用来设置属性是否可以写
