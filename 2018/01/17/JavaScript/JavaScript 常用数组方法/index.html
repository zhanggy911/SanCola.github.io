JavaScript 常用数组方法

发表于 2018-01-17 | 分类于 JavaScript

concat()

  • 描述
    • 连接两个或多个数组
  • 语法
    • arr.concat(arr1,arr2….arrN)
  • 参数
    • arrN，必需，可以是具体的值，也可以是数组对象，个数不限
  • 返回值
    • 修改原数组
    • 返回被连接数组的一个新副本
  • 实例

join()

  • 描述
    • 把数组中的所有元素放入一个字符串，并用指定的分隔符分隔
  • 语法
    • arr.join(separator)
  • 参数
    • separator，可选，指定分隔符，如果省略，则使用逗号分隔
  • 返回值
    • 不修改原数组
    • 返回一个字符串，将原数组的所有元素连接起来，并插入分隔符
  • 实例

pop()

  • 描述
    • 删除并返回数组的最后一个元素
  • 语法
    • arr.pop()
  • 参数
    • 无
  • 返回值
    • 修改原数组，删除原数组的最后一位，length 减一
    • 返回该数组被删除的最后一个元素
    • 如果数组已经为空，则不改变数组，并返回 undefined
  • 实例

push()

  • 描述
    • 数组末尾添加一个或多个元素，并返回新的长度
  • 语法
    • arr.push(ele1,ele2…eleN)
  • 参数
    • ele1，必须，要添加到数组的元素
    • ele2，eleN，可选，可以添加多个元素
  • 返回值
    • 修改原数组
    • 返回添加指定值后的数组 length
  • 实例

shift()

  • 描述
    • 将数组的第一个元素删除，并返回被删除的元素
  • 语法
    • arr.shift()
  • 参数
    • 无
  • 返回值
    • 修改原数组
    • 返回数组被删除的第一个值，如果数组是空的，不会对原数组产生作用，并返回 undefined
  • 实例

unshift()

  • 描述
    • 向数组的开头添加一个或多个元素，并返回添加元素后的数组 length
  • 语法
    • arr.unshift(ele1,ele2…eleN)
  • 参数
    • ele1，必需，向数组添加的第一个元素
    • ele2，eleN，可选，允许向数组添加多个元素
  • 返回值
    • 修改原数组
    • 返回添加元素后的数组 length
  • 实例

slice()

  • 描述
    • 从已有的数组中返回选定的元素
  • 语法
    • arr.slice(start,end)
  • 参数
    • start，必选，规定从何处开始选取，数组片段开始的数组下标
      • 选取的数组片段包含该起始位置的数组元素
      • 如果是负数，表示从数组的尾部开始算起，-1表示最后一位，-2表示倒数第二位
    • end，可选，规定从何处结束选取，数组片段结束的数组下标
      • 选取的数组片段不包含该结束位置的数组元素
      • 如果未指定该参数，那么将从规定的 start 点开始截取，直到数组结束
      • 如果指定的 end 为负数，表示从数组的尾部开始算起
  • 返回值
    • 不修改原数组
    • 返回选取到的新数组，如果选取不到，则返回空数组
  • 实例

splice()

  • 描述
    • 在数组中 添加/删除 元素，并返回被删除的项目
  • 语法
    • arr.splice(index,howmany,item1…itemN)
  • 参数
    • index，可选，整数，规定添加/删除项目的位置
      • 删除的数组元素包含该位置的数组元素
      • 使用负数可从数组结尾处规定位置
      • 缺省该参数会直接返回空数组
    • howmany，可选，需要删除项目的数量
      • 如果设置为0，则不会删除项目
      • 如果不设置该项，则会从规定的 index 处向后删除所有数组中的项目
    • itme1，itemN，可选，向数组添加的新项目
      • 添加的项目位于指定的起始位置之前
  • 返回值
    • 修改原数组
    • 如果删除了元素，返回包含被删除项目的新数组
  • 实例

reverse()

  • 描述
    • 颠倒数组中的元素顺序
  • 语法
    • arr.reverse()
  • 参数
    • 无
  • 返回值
    • 修改原数组
    • 返回颠倒元素后的数组
  • 实例

sort()

  • 描述
    • 对数组元素进行排序
  • 语法
    • arr.sort(sortby)
  • 参数
    • sortby，可选，规定排序顺序，必需是函数
      • 如果未规定该参数，将对数组元素按照字符编码顺序进行排序，必要时需要先转换成字符串数组
      • 如果需要按照其他顺序排序，需要制定排序函数
  • 返回值
    • 修改原数组
    • 返回排序后的数组
  • 实例

filter()

  • 描述
    • 使用指定的函数测试所有元素，并返回通过测试的元素组成的新数组
    • 不兼容 IE8及以下
  • 语法
    • arr.filter(callback , thisArg)
  • 参数
    • callback
      • 用来测试数组的每个元素的函数，调用时可以使用参数（element,index,array）
    • thisArg
      • 可选，执行callback时的用于this的值
      • 不指定时，callback参数的this指向window，严格模式下为undefined
  • 返回值
    • 不修改原数组
    • 返回原数组通过测试的元素组成的新数组
  • 实例

forEach()

  • 描述
    • 依次对数组中的值执行依次 callback 函数
  • 语法
    • arr.forEach(callback,thisArg)
  • 参数
    • callback
      • 为数组中的每个元素执行的函数，该函数接收3个参数
        • currentValue：数组中正在处理的当前元素
        • index：数组中正在处理的当前元素的索引
        • array：forEach()方法正在操作的数组
    • thisArg
      • 可选，执行callback时的用于this的值
      • 不指定时，callback参数的this指向window，严格模式下为undefined
  • 返回值
    • 修改原数组
    • 永远返回undefined
    • 不可跳出，不可链式调用
  • 实例

map()

  • 描述
    • 为数组的每一个值调用 callback 函数，并将每次执行 callback 取得的值组成新数组并返回
  • 语法
    • arr.map(callback,thisArg)
  • 参数
    • callback
      • 为数组中的每个元素执行的函数，该函数接收3个参数
        • currentValue：数组中正在处理的当前元素
        • index：数组中正在处理的当前元素的索引
        • array：map()方法正在操作的数组
    • thisArg
      • 可选，执行callback时的用于this的值
      • 不指定时，callback参数的this指向window，严格模式下为undefined
  • 返回值
    • 不修改原数组
    • 返回执行 callback 函数后组成的新数组
  • 实例

indexOf()

  • 描述
    • 返回被找到的元素在数组中的索引位置，没找到返回-1
  • 语法
    • arr.indexOf(element,formIndex)
  • 参数
    • element：要查找的元素
    • fromIndex：开始查找的位置
      • 值大于数组 length 长度，返回-1
      • 负值表示从数组最后开始查找，-1代表最后一位
  • 返回值
    • 找到返回元素索引，找不到返回-1
  • 实例

toString()

  • 描述
    • 将数组中的所有元素使用逗号分隔，并返回由所有数组元素组成的字符串
  • 语法
    • arr.toString()
  • 参数
    • 无
  • 返回值
    • 不修改原数组
    • 返回字符串
  • 实例

toLocaleString()

  • 描述
    • 为数组中的元素调用 toLocalString 方法转换成字符串并返回
  • 语法
    • arr.toLocalString()
  • 参数
    • 无
  • 返回值
    • 不修改原数组
    • 返回字符串
  • 实例
