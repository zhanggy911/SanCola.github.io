JavaScript 常用时间方法

发表于 2018-01-17 | 分类于 JavaScript

Date()

  • 描述：返回当天的日期和时间
  • 语法：Date()
  • 返回值：字符串

getDate()

  • 描述：返回月份中的某一天
  • 语法：d.getDate()
  • 返回值：数字（1-31）

getDay()

  • 描述：返回星期中的某一天
  • 语法：d.getDay()
  • 返回值：数字（0-6，周日到周六）

getMonth()

  • 描述：返回月份的数字
  • 语法：d.getMonth()
  • 返回值：数字（0-11，1月到12月）

getFullYear()

  • 描述：返回年份的4位数字
  • 语法：d.getFullYear()
  • 返回值：数字

getHours()

  • 描述：返回小时
  • 语法：d.getHours()
  • 返回值：（0-23）

getMinutes()

  • 描述：返回分钟
  • 语法：d.getMinutes()
  • 返回值：（0-59）

getSeconds()

  • 描述：返回秒数
  • 语法：d.getSeconds()
  • 返回值：(0-59)

getMilliseconds()

  • 描述：返回当前毫秒数
  • 语法：d.getMilliseconds()
  • 返回值：（0-999）

getTime()

  • 描述：返回自1970年1月1日至当前的毫秒数
  • 语法：d.getTime()
  • 返回值：数字

toString()

  • 描述：将 Data 对象转换为字符串并返回结果
  • 语法：d.toString()
  • 返回值：字符串

toLocalString()

  • 描述：将 Data 对象转换为字符串并返回结果
  • 语法：d.toLocalString()
  • 返回值：字符串
