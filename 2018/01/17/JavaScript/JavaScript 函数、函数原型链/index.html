JavaScript 函数、函数原型链

发表于 2018-01-17 | 分类于 JavaScript

函数声明方式

函数声明

函数表达式

Function构造函数

  • Function 是一个构造函数，用来创建一个函数，Function() 括号内部的函数体必需是字符串；
  • 解决Function的函数体过长的问题
    • 字符串拼接
    • 使用模板引擎
  • 函数不能重名，后面的会覆盖前面的
  • 形参与实参可以不一致，多出来的实参会废掉
  • 如果实参没有赋值，那函数形参的值是undefined
  • return会返回运算的值，并跳出函数运算
    • 如果return后面有值，那么返回值
    • 如果return后面没有值，那么返回undefined，后面的代码不执行
    • 如果没写return，返回undefined

函数分类

构造函数

  • 构造函数也是函数，一般用来来初始化对象
  • 构造函数的特点
    • 首字母大写
    • 一般和new关键字一起使用
    • 不需要手动写返回值
  • 构造函数的执行过程
    • 使用 new 关键字创建对象
    • 调用构造函数
    • 将构造函数内部的 this 指向 new 创建出来的对象
    • 在构造函数内部使用 this 为对象新增成员
    • 返回该对象
  • 构造函数的使用注意事项
    • 返回值：不需要手动写返回值，如果写了
      • 如果是值类型的，将不会对返回值有任何的影响
      • 如果是引用类型的，则会将默认返回值进行替换
    • 如果把构造函数当做普通函数调用，this 会指向 window，返回值如果不写，会是 undefined
  • 传统构造函数的问题
    • 资源浪费
    • 解决办法：将重复的函数的声明放在最外面，在构造函数的内部将地址赋值
  • 特殊情况：
    • new Object('数据类型')，创建出来的就是什么类型

回调函数

  • 在 JavaScript 中，回调函数具体的定义为：函数A作为参数传递到另一个函数B中，并且这个函数B执行函数A。我们就说函数A叫做回调函数。如果没有名称(函数表达式)，就叫做匿名回调函数。
  • 使用场景：动态加载js文件后执行回调，加载 iframe 后执行回调，ajax 操作回调等

自调用函数（IIFE）

  • js中没有块级作用域，可以使用自调用函数来隔离作用域，避免污染，或者截断作用域链，避免闭包造成引用变量无法释放
  • 利用立即执行特性，返回需要的业务函数或对象，避免每次通过条件判断来处理
  • IIFE也常用于写惰性载入，因为函数被执行引擎以同步的方式立即执行了，所以在之后的代码访问这个变量的时候，可以直接返回计算后的筛选结果

函数的原型链

  • 如果将函数当做一个对象对待的话，那么他也是由构造函数创建出来的，所以这个函数对象，也会有相应的原型链
  • js中所有的函数的构造函数都是Function, 包括Function本身
  • Function是自己的构造函数，也是自己的实例
  • Function构造函数在原型图中可以担任两个角色，一个是构造函数，一个是实例
  • 当作为构造函数的时候，访问原型使用 Function.prototype

  • 但作为实例的时候，访问原型使用Funciton.__proto__

  • 完整原型链

  • instanceof 关键字
    • 作用：判断构造函数的原型是不是在对象的原型链上
    • 语法：对象 instanceof 构造函数
  • Object 和 Function 的关系
    • 可以通过instanceof关键字来确定两者的关系， Object和Function互为实例

函数对象成员

函数对象成员

arguments

  • 函数内部的一个对象，是一个伪数组，在函数调用的时候，会将函数的实参，依次的存入这个伪数组中
  • arguments.length
    • 表示函数在调用的时候，传入的实参的个数
  • arguments.callee
    • 指向这个函数本身，一般在匿名函数进行递归时使用

caller

  • 函数的调用环境，如果是在全局，则为null，如果是在其他函数当中调用的，那么caller为所在的那个函数

length

  • 函数的形参的个数

name

  • 函数的名称 函数声明和函数表达式的函数名为正常的函数名，new Function 创建出来的函数名为 anonymous
  • 函数表达式写两个 name 时，var a = function b(){}, b 只能在函数内部使用。且 name 属性与 b 相同
  • 函数体内部的任何参数与函数对象的成员都没有关系；

静态成员以及实例成员

  • 静态成员
    • 通过构造函数访问的成员就是静态成员
  • 实例成员
    • 通过实例去访问的成员就是实例成员
  • 一般将工具方法，也就是类似于 $.ajax，设置为静态成员
  • 一般将和对象相关的成员，类似于.css() 这种方法，设置为实例成员
