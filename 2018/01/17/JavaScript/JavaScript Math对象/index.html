JavaScript Math对象

发表于 2018-01-17 | 分类于 JavaScript

Math对象没有构造函数，也无需创建

1. Math对象属性

E

  • 描述：算数常量e，即自然对数的底数

LN2

  • 描述：2 的自然对数

LN10

  • 描述：10 的自然对数

LOG2E

  • 描述：2 为底 e 的对数

LOG10E

  • 描述：10 为底 e 的对数

PI

  • 描述：圆周率π

SQRT1_2

  • 描述：2 的平方根的倒数

SQRT2

  • 描述： 2 的平方根

2. Math对象常用方法

abs(x)

  • 描述：返回一个数得绝对值

pow(x,y)

  • 描述：返回x 的 y 次方

sqrt()

  • 描述：返回1个数的平方根

ceil(x)

  • 描述：向上取整

floor(x)

  • 描述：向下取整

max(x,y)

  • 描述：返回较大的值

min(x,y)

  • 描述：返回较小的值

random()

  • 描述：返回0-1之间的一个随机数

round(x)

  • 描述：将值四舍五入

sin(x)

  • 描述：返回角的正弦，x 是弧度表示的角

cos(x)

  • 描述：返回角的余弦

tan(x)

  • 描述：返回角的正切
