switch语句

  • 代码格式
    • case代码块后面必须加入break
    • switch后面的变量数据类型与case后面值的数据类型必须保持一致
  • 与 if语句 的区别
    • if语句的条件表达时的结果是一个boolean类型，switch语句可以不是
    • switch适合一个值和多个值是否相等的情况，且switch语句判断的是全等（===）

delete 关键字

  • 语法
    • delete 变量名/对象.属性名
  • 返回值
    • boolean类型，但是并不可行
  • 作用
    • 删除未使用var声明的变量，使用var声明的无法删除
    • 删除对象的属性
  • 注意
    • 使用delete删除数组的某个值时，相当于这个值消失了，但是占位还在，length不变
