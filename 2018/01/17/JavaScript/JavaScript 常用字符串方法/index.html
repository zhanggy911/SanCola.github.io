JavaScript 常用字符串方法

发表于 2018-01-17 | 分类于 JavaScript

由于字符串值的不可变性，所有的字符串方法都不能改变原有字符串。只返回新的字符串

如果需要改变字符串，可以重新赋值，str = str.slice(2 ,4 )

charAt()

  • 描述
    • 返回指定位置的字符串
  • 语法
    • str.charAt(index)
  • 参数
    • index，必需，表示字符串中某个位置的数字，既字符在字符串中的下标
  • 返回值
    • 返回的是长度为1的字符串
  • 实例

charCodeAt()

  • 描述
    • 返回指定位置的字符串的 Unicode 编码
  • 语法
    • str.charCodeAt(index)
  • 参数
    • index，必需，表示字符串中某个位置的数字，既字符在字符串中的下标
  • 返回值
    • 返回的是Unicode 编码
  • 实例

indexOf()

  • 描述
    • 返回指定的字符串值在字符串中首次出现的位置
    • 对大小写敏感
  • 语法
    • str.indexOf(value,fromIndex)
  • 参数
    • value：必需，规定的要检索的字符串
    • fromIndex：可选，规定在字符串中开始检索的位置
      • 省略该参数将从开始处检索字符串
  • 返回值
    • 该字符串在数组中的位置，找不到返回-1
  • 实例

search()

  • 描述
    • 检索字符串中指定的子字符串，或检索与正则表达式相匹配的子字符串
  • 语法
    • str.search(regexp/substr)
  • 参数
    • regex：可以是正则表达式，也可以是指定的子串
  • 返回值
    • 第一个匹配到的子串的起始位置
    • 不能使用正则表达式的全局模式
  • 实例

replace()

  • 描述
    • 在字符串中用一些字符替换另一些字符，或替换一个与正则表达式匹配的子串
  • 语法
    • str.replace(regexp/substr,replacement)
  • 参数
    • regexp/substr，必需，规定子字符串或要替换的模式的 regExp对象
    • replacement，必需，规定替换文本或生成替换文本的函数
  • 返回值
    • 一个完成了替换的字符串
    • 如果没有启用全局模式，那么只替换第一个匹配的子串
  • 实例

match()

  • 描述
    • 按照指定的正则规则，检索字符串，并返回由符合规则的字符串组成的数组
  • 语法
    • str.match(value,regexp)
  • 参数
    • value：必需，需要检索的值
    • regex：规定要匹配的正则规则，如果该参数不是 regexp，则先将该参数传递个正则构造函数
  • 返回值
    • 返回一个由符合规则的字符串组成的数组
    • 如果正则对象没有启用全局模式，那么只在 str 中执行一次匹配
    • 匹配不到值返回 null
  • 实例

slice()

  • 描述
    • 提取字符串中的某个部分并返回
  • 语法
    • str.slice(start,end)
  • 参数
    • start：必需，指定的起始下标（包含该下标表示的元素），如果是负数，表示从最后面算起
    • end：可选，指定的结尾下标（不包含该下标表示的元素），如果是负数，则从尾部开始算起
      • 如果忽略，则返回从指定的 start 开始获取的所有字符串
  • 返回值
    • 返回获取到的字符串，无法获取返回空字符串
  • 实例

substring()

  • 描述
    • 提出字符串中指定位置的字符
  • 语法
    • str.substring(start,stop)
  • 参数
    • start，必需，一个正整数，指定开始提取的位置
    • end，可选，一个正整数，指定结束提取的位置
    • 与 slice() 不同的是不能使用负值
  • 返回值
    • 获取到的字符串，获取不到返回空字符串
  • 实例

split()

  • 描述
    • 把一个字符串按照指定的方式分隔成字符串数组，该方法与 arr.prototype.join() 方法相反
  • 语法
    • str.split(separator,howmany)
  • 参数
    • separator：必需，字符串或正则对象，从该参数指定的地方分隔
    • howmany：可选，指定返回数组的最大长度，不指定该参数则不会考虑返回的数组长度
  • 返回值
    • 一个字符串数组，无法分隔返原字符串

  • 实例

toLowerCase()

  • 描述
    • 字符串转换为小写
  • 语法
    • str.toLowerCase()
  • 参数
    • 无
  • 返回值
    • 转换完成的字符串
  • 实例

toUpperCase()

  • 描述
    • 字符串转换为大写
  • 语法
    • str.toUpperCase()
  • 参数
    • 无
  • 返回值
    • 转换完成的字符串
  • 实例
