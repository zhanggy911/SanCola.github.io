JavaScript 概念 数据类型 变量

发表于 2018-01-17 | 分类于 JavaScript

概念

运行在客户端浏览器的脚本编程语言

介绍

  • 弱类型语言
  • 属于脚本语言，不需要编译，边解析边运行

组成

  • ECMAScript
  • DOM
  • BOM

书写位置

  • 内嵌式写法 — 推荐将 js 代码放置在 body 标签内部的最后面
  • 外连式写法 — 新建 js 文件，通过 script 标签引入

script标签属性

  • src — 引用位置
  • async — 异步加载
  • defor — 异步加载
  • sync — 同步加载

注释

页面输出消息的几种方法

  • alert()
    • 数字类型／变量不需要引号，字符串需要加 ‘’ 号
  • console.log()
    • 向控制台输出一段信息
  • console.dir()
    • 在控制台中打印出元素具有哪些变量
  • document.write()
    • 直接在页面中输出信息，可以直接写html标签，但是需要添加 ‘ ’
  • confirm()
    • 弹出选择框，可以配合条件判断使用
  • prompt()
    • 弹出输入框，让用户输入信息

数据类型

数据类型分类

简单类型

  • Number — 数值类型
  • String — 字符串类型
  • Boolean — 布尔类型
  • Undefined — 未定义／找不到
  • Null — 不存在

复杂类型

  • 内置对象
    • function
    • Math
  • 构造函数
    • Object — 对象
    • Array — 数组
    • Date — 时间

Number

数字类型的表示方法

  • 十进制 — 0-9
  • 十六进制 — 0-9 a-f
  • 八进制 — 0-7

浮点数

  • 科学计数法，例子：5e-324（5*10的-324次方）
  • 浮点数值的最高精度为17位小数
  • 不要判断浮点数值是否相等，非要判断的话，需要原数值＊10/100，取整后再进行比较
  • 最大值：Number.MAX_VALUE
  • 最小值：Number.MIN_VALUE
  • 无穷大／小：Infinity／-Infinity

String

“ ” 里的内容即是字符串。可以是 ”” 也可以使用 ‘ ’

转译符

  • 如果需要在字符串内加 ” ” ，可以切换使用单双引号
  • 也可以使用转译符：\’（转译单引号） \” （转译单引号）
  • \n 字符串换行
  • \t 缩进
  • \b 空格
  • \r 回车
  • \ 斜杠

length 属性

字符串的不可变

字符串一旦创建，那么它的值就是不可变的，想要改变值，需要在内存中重新开辟新的空间来存储新的值，原来的值／地址仍然会存在一段时间，直到被内存清理机制清除掉，所以字符串赋值时尽量不要用循环语句赋值，应当使用数组赋值

Boolean

Boolean只有两个值，true ／ false，通常用于判断

Undefined

如果一个变量没有赋值，那么这个变量的默认值就是 undefined，其对应的数据类型是 Undefined

Undefined 类型不会使浏览器报错

Null

  • 既是简单类型，又是复杂类型（一般还是归类在简单类型中）
  • 简单类型理由：undefined == null
  • 复杂类型理由：typeof null

数据类型的判断

typeof(值)

  • 此时控制台现实的不再是输入的值，而是判断出值的类型（string，number等）
  • typeof不是函数，而是关键字，后面的括号是为了提升运算优先级，可以不写
  • typeof判断复杂类型的时候，只能判断出 Object 和 function

isNaN(值)

  • 判断当前的数值是否是NaN，如果是数字，返回false，如果不是数字，返回的是true
  • NaN: not a number 结果为非数字。用来表示数值的一种非正常状态

引用类型和值类型

堆、栈概念

  • 这个概念只是用来帮助理解，实际上js中应该并不存在堆栈的概念，即使存在，也是在浏览器的js引擎中

值类型

  • 直接存储数据本身的数据，就是值类型（就是简单类型）

引用类型

  • 直接存储的是数据的地址，数据的值单独存储的数据，就是引用类型（就是复杂类型）

值类型和引用类型的赋值特征

值类型
  • 值类型的数据在赋值的时候，会将存储的数据复制一份交给新的变量
  • 两个变量值相等，但是存储的空间不一样，修改其中一个，另外一个不会受影响
引用类型
  • 引用类型的数据在赋值的时候，会将对象的地址复制一份交给新的变量
  • 此时两个变量同时指向同一个对象，通过其中一个变量对对象进行修改，另外一个变量也会受到影响

值类型与引用类型的比较

  • 不能直接比较，在比较运算发生时，引用类型会先调用.valueOf()方法，返回的值如果可以参与运算，就可以进行比较
  • 如不仍然不能进行运算，会继续调用.toString()方法，用返回值进行比较

引用类型之间的比较

  • 引用类型之间的比较是先比较引用类型的地址

变量

定义

  • 变量是用来存储数据的
  • 一个变量只能保存一个值，且只能保存最后一次赋值的结果

变量声明

  • var — 变量声明的关键字
  • 可以同时声明多个变量
  • 赋值时，等号右侧允许运算
  • 未使用 var 声明的变量，都是全局变量，无论该变量出现在全局作用域还是局部作用域
  • 声明一个全局变量，相当于给window对象添加了一个属性

变量命名规则

  • 变量只能由字母、数字、下划线、$构成
  • 数字不能出现在变量名称最前面
  • 使用驼峰命名法： userName
  • 区分大小写
  • 不能使用 javascript 的保留字段及关键字

变量类型

  • 变量类型由赋值决定

变量值 undefined、null 的判断

  • 变量的值在什么时候是undefined
    • 声明之后未赋值
    • 手动赋值为undefined
    • 对象属性不存在，此属性的值为undefined
  • 变量的值在什么时候是null
    • 变量不能自动为null，除非手动赋值
    • 当为其它开发者提供一个返回对象的方法或者函数的时候，会有手动赋值为null的情况
    • 当函数中创建对象失败，或者获取对象失败，无法正常返回对象的时候，就应该返回null
