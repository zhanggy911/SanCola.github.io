JavaScript 循环、数组

发表于 2018-01-17 | 分类于 JavaScript

循环

while 循环

  • 代码格式
  • 执行过程
    • 如果条件表达式为 true，那么循环体中的代码会一直执行
    • 如果为 false，那么该循环代码不再执行

do …while 循环

  • 代码格式
  • 执行过程
    • 先执行 do 中的代码体
    • 执行完 do 的代码体后，再判断条件是否成立
    • 如果成立，则继续执行，如果不成立，do 中的代码不再执行
  • 与 while 循环的区别
    • while 语句是先判断，后执行，do …while 语句是先执行再判断
    • 如果条件不成立的话，do while 要比 while 多执行一次

for循环

  • 代码格式

  • 执行过程

    • 首先进行变量初始化，判断条件是否成立
    • 如果条件成立，那么执行for 循环中的代码体，然后进行变量自增自减
    • 继续判断条件是否成立，如果成立则继续代码体，不成立则代码不再执行

  • 与 while／do...while 循环的区别

    • 当循环次数未知时，使用 while / do...while 循环
    • 当循环次数已知时，使用 for 循环

for…in循环

  • 一般用于遍历对象

  • 在 for...in 循环中不能使用 . 语法访问对象

  • in关键字

    • 在 for...in 循环中遍历所有的键
    • 判断对象是否可以访问某个属性

break语句

  • 循环体中，当代码遇到 break 语句时，循环体代码立即结束（仅限当前循环）

continue语句

  • 循环体中，执行到 continue 语句时，会结束本次循环，进入到下一次循环中

数组

数组与变量的区别

  • 变量只能保存一个值
  • 数组可以保存多个值（任何数据类型）

数组的定义

  • 通过对象的方式：var ary=new Array() 创建一个空数组
  • 直接创建数组：var ary1=[] 创建一个空数组

创建一个有默认值的数组

数组赋值/取值

  • 数组赋值的时候需要通过下标（索引）设置值
  • 数组的下标是从0开始的，0代表第一个，以此类推

数组的遍历

  • 数组的 length 属性
    • arr.length 的值是动态改变的
    • 修改 arr.length 值会改变数组的值

伪数组

  • 不具备数组方法，可以遍历，有length属性,但是length属性需要手动添加，不会自动更新
