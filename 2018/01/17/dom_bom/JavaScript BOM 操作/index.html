JavaScript BOM 操作

发表于 2018-01-17 | 分类于 JavaScript

概念

  • BOM 是 browser object model 的缩写，简称浏览器对象模型
  • BOM 提供了独立于内容而与浏览器窗口进行交互的对象
  • 由于 BOM 主要用于管理窗口与窗口之间的通讯，因此其核心对象是 window
  • BOM 由一系列相关的对象构成，并且每个对象都提供了很多方法与属性
  • BOM 缺乏标准，JavaScript 语法的标准化组织是 ECMA，DOM 的标准化组织是 W3C
  • 全局变量相当于 window 对象的一个属性

常用方法

  • window.alert() — 弹出提示信息
  • window.confirm() — 弹出选择框
  • window.prompt() — 弹出输入框
  • window.onScroll() — 页面滚动事件
  • window.scrollTo() — 页面滚动至
  • window.onresize() — 页面尺寸改变
  • window.onload() — 页面加载完成
  • window.open(url, target, param) — 打开新页面
    • url：要打开的地址
    • target：新窗口的位置（_blank / _self）
    • param：新窗口的参数（width / height 等）
  • window.close() — 关闭 window 窗口（谁调用关闭谁）

常用属性

window.location

  • window.location.assign(url) — 跳转到新页面
  • window.location.replace(url) — 跳转到新页面，无法退回
  • window.location.reload() — 刷新页面
  • window.location.hash — 哈希值
  • window.location.host — 主机
  • window.location.hostname — 主机名
  • window.location.pathname — 路径名
  • window.location.port — 端口号
  • window.location.protocol — 协议
  • window.location.search — 参数

window.navigator

  • window.navigator.userAgent — 获取一些浏览器信息
  • window.navigator.platfrom — 运行平台信息
  • window.navigator.onLine — 网络连接
  • window.navigator.geolocation — 定位信息

window.histroy

  • window.history.go(1) — 表示前进一个页面
  • window.history.go(-1) — 表示后退一个页面
  • window.history.forward() — 表示前进
  • window.history.back() — 表示后退

window.screen

  • window.screen.width) — 宽度
  • window.screen.height) — 高度
  • window.screen.availWidth) — 可用宽度
  • window.screen.availHeight) — 可用高度

web 本地存储功能

  • window.sessionStorage
    • 生命周期为关闭浏览器窗口
    • 在同一个窗口下数据可以共享
  • window.localStorage
    • 永久生效，除非手动删除
    • 可以多窗口共享
  • 相关属性
    • setItem(key, value) — 设置存储内容
    • getItem(key) — 读取存储内容
    • removeItem(key) — 删除键值为key的存储内容
    • clear() — 清空所有存储内容
    • key(n) — 以索引值来获取存储内容

定时器

延时定时器

  • setTimeout — 设置定时器
  • clearTimeout清除定时器

间歇定时器

  • setInterval() — 设定定时器
  • clearInterval() — 清除定时器
