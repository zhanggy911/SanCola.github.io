JavaScript DOM 事件操作

发表于 2018-01-17 | 分类于 JavaScript

绑定、移除事件

绑定事件

  • on
    • 语法：事件源.事件 = function(){ 事件处理程序 };
    • 一个元素只能绑定一个事件
  • addEventListener
    • 语法：element.addEventListener('事件类型'，'事件处理函数'，'是否使用捕获')
    • 一个元素可以绑定多个事件，依次执行
    • addEventListener 通过回调函数的参数获取
    • addEventListener 中 this 指向当前 dom 对象
  • attachEvent()
    • 在 IE 上需要使用 attachEvent()
    • 语法：attachEvent(type, handler)
    • attachEvent 通过 window.event 获取 事件对象
    • attachEvent 中 this 指向 window

移除事件

  • 方式1：element.onclick = null()
  • 方式2：removeEventListener()
  • 如果注册的时候使用的是匿名函数，则无法移除
  • 通过 innerHTML 清空内容的时候，事件相关函数不会被销毁，因此建议使用 removeChild 来清除

event对象

在触发事件时，会产生一个事件对象 event，这个对象中包含着与事件有关的信息。所有浏览器都支持 even t对象，但支持的方式不同。比如鼠标操作时候，会添加鼠标位置的相关信息到事件对象中。

事件的三个阶段

捕获阶段

  • 当某个元素触发某个事件（如onclick），顶层对象document就会发出一个事件流，随着DOM树的节点向目标元素节点流去，直到到达事件真正发生的目标元素。此过程中事件监听函数不会被触发。

目标阶段

  • 当到达目标元素之后，执行目标元素该事件相应的处理函数。如果没有绑定监听函数，那就不执行。

冒泡阶段

  • 从目标元素开始，往顶层元素传播。途中如果有节点绑定了相应的事件处理函数，这些函数都会被依次触发
  • 阻止事件冒泡
    • W3C： event.stopPropagation()
    • IE：event.cancelBubble = true

事件代理

当需要对很多元素添加事件时，可以通过将事件委托给其父元素来触发处理函数

  • 事件代理优点
    • 不需要为每个元素都添加监听函数，减少需要管理的函数
    • 不需要因为元素的改动而修改事件绑定
    • 减少 JavaScript 和 DOM 节点之间的关联，也就减少了因循环引用而带来的内存泄漏发生的概率

常用事件

常用鼠标事件

  • onclick — 点击事件
    • 当使用点击跳转时，只能写return false
  • onmouseover — 鼠标移入
  • onmouseout — 鼠标移出
  • onmousemove — 鼠标移动
  • onmouseup — 鼠标抬起
  • onmousedown — 鼠标按下
  • onmouseleave — 鼠标离开范围

常用键盘事件

  • onkeypress — 输出的对应键盘按键的ASCII码表
  • onkeydown — 键盘按下，输出键盘码
  • onkeyup — 键盘抬起，输出键盘码

焦点事件

  • onfocus — 获取焦点
  • onblur — 失去焦点 (不是所有的标签都有这个事件)

全屏幕

  • requestFullScreen() — 打开全屏幕
  • cancleFullScreen() — 关闭全屏幕
  • 有兼容问题

文件拖拽

  • 拖拽元素
    • ondrag — 调用数次
    • ondragstart — 调用一次，在开始时
    • ondragleave — 调用一次，在离开元素本来范围时
    • ondragend — 调用一次，在结束时
  • 目标元素
    • ondragenter — 调用一次，在进入范围时
    • ondragover — 调用数次，在停留范围时
    • ondrop — 调用一次，在松开鼠标时
    • ondragleave — 调用一次，在离开范围时
  • 如果想要调用 ondrop 事件，必须在 ondragover 事件中使用参数阻止浏览器的默认行为

文件读取

  • 使用 js 对象 FileReader
  • 配合 File 对象可以指定读取本地文件,可以读取选取的文件,也可以读取 DataTransfor
  • 获取文件
    • 由于 File 标签支持多文件上传,所以我们想要获取到选取的文件,需要配合索引
  • 读取文件
    • 读取文件需要使用 js 对象 FileReader

数据传递

  • dataTransfor — 数据改变事件

视频操作

  • video.play() — 开始
  • video.pause() — 暂停
  • video.onprogress() — 开始时执行
  • video.ontimeupdate() — 播放时一直执行
  • video.currentTime() — 当前时间
  • video.duration() — 视频总长

节流阀概念

当事件执行的过于频繁时，可以使用定时器来减少函数的执行次数
