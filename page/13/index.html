JavaScript 常用字符串方法

发表于 2018-01-17 | 分类于 JavaScript

由于字符串值的不可变性，所有的字符串方法都不能改变原有字符串。只返回新的字符串

如果需要改变字符串，可以重新赋值，str = str.slice(2 ,4 )

charAt()

  • 描述
    • 返回指定位置的字符串
  • 语法
    • str.charAt(index)
  • 参数
    • index，必需，表示字符串中某个位置的数字，既字符在字符串中的下标
  • 返回值
    • 返回的是长度为1的字符串
