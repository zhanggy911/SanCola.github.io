JavaScript BOM 操作

发表于 2018-01-17 | 分类于 JavaScript

概念

  • BOM 是 browser object model 的缩写，简称浏览器对象模型
  • BOM 提供了独立于内容而与浏览器窗口进行交互的对象
  • 由于 BOM 主要用于管理窗口与窗口之间的通讯，因此其核心对象是 window
  • BOM 由一系列相关的对象构成，并且每个对象都提供了很多方法与属性
  • BOM 缺乏标准，JavaScript 语法的标准化组织是 ECMA，DOM 的标准化组织是 W3C
  • 全局变量相当于 window 对象的一个属性

常用方法

  • window.alert() — 弹出提示信息
  • window.confirm() — 弹出选择框
  • window.prompt() — 弹出输入框
  • window.onScroll() — 页面滚动事件
  • window.scrollTo() — 页面滚动至
  • window.onresize() — 页面尺寸改变
  • window.onload() — 页面加载完成
  • window.open(url, target, param) — 打开新页面
    • url：要打开的地址
    • target：新窗口的位置（_blank / _self）
    • param：新窗口的参数（width / height 等）
  • window.close() — 关闭 window 窗口（谁调用关闭谁）

JavaScript DOM 动画

发表于 2018-01-17 | 分类于 JavaScript

动画原理公式

  • 动画原理公式： leader = leader + step
  • leader 表示盒子当前位置
  • step 表示步长
  • 让 setInterval 不断执行某个函数修改盒子的位置属性最后就实现了动画的效果

缓动动画

  • 匀速动画公式
    • step = 定值
    • leader = leader + step
  • 缓动动画公式
    • step = ( target - leader ) / 10
    • leader = leader + step

JavaScript DOM 获取元素、节点操作

发表于 2018-01-17 | 分类于 JavaScript

基本概念

  • DOM定义
    • Document Object Model 文档对象模型
    • 就是把 HTML 文档模型化，当作对象来处理
  • 内容概念
    • Document（文档）
      • 就是指 HTML 或者 XML 文件
    • Node（节点）
      • HTML文档中的所有内容都可以称之为节点，常见的节点有
        • 元素节点
        • 属性节点
        • 文本节点
        • 注释节点
    • Element（元素）
      • HTML 文档中的标签可以称为元素
  • 结构概念
    • 父节点：当前节点的父级
    • 子节点：当前节点的子级
    • 兄弟节点：和当前节点同属一个父级的节点
