ES6 Promise对象

发表于 2018-01-18 | 分类于 JavaScript

Promise 是异步编程的一种解决方案，比传统的解决方案——回调函数和事件——更合理和更强大。它由社区最早提出和实现，ES6 将其写进了语言标准，统一了用法，原生提供了 Promise对象。

概念

  • Promise 对象
    • Promise 是一个对象，从它可以获取异步操作的消息
    • 使用 Promise 对象，可以将异步操作以同步操作的方式表达出来，避免嵌套
  • Promise 对象状态
    • Pending — 进行中
    • Resolved — 已完成
    • Rejected — 已失败
  • Promise 对象状态特性
    • 对象状态不受外界影响
      • 只有异步操作的结果可以决定当前是哪一种状态
    • 对象状态一旦改变，就不会再次改变
      • 对象状态改变只用两种情况
        • Pending — Resolved
        • Pending — Rejected
  • 注意
    • 无法取消 promise对象，一旦创建就会立即执行，无法取消
    • 不设置回调函数，promise 内部抛出的出错误不会反应到外部
    • 当 promise对象处于 pending 状态时，无法获知目前进展到哪一个阶段
