canvas 绘制文本、图片、变换

发表于 2018-01-17 | 分类于 HTML

绘制文本

  • ctx.strokeText(text, x, y, maxWidth) — 空心文本
    • text — 要绘制的文本
    • x, y — 文本的绘制位置，这个点默认在文本的左下角的基线位置
    • maxWidth — 限制文本的最大宽度 如果文本长度大于maxWidth 会被压扁
  • ctx.fillText(text, x, y, maxWidth) — 实心文本
    • text — 要绘制的文本
    • x, y — 文本的绘制位置，这个点默认在文本的左下角的基线位置
    • maxWidth — 限制文本的最大宽度 如果文本长度大于maxWidth 会被压扁
  • ctx.measureText(text)
    • 返回一个对象，对象中包含一个 width 属性，表示文本在当前上下文字体下显示出来的宽度
      strokeText / fillText 具备独有路径，之后的任何操作，不会对其产生影响
