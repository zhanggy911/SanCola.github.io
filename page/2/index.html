yarn 包管理工具

发表于 2018-01-18 | 分类于 tools

Yarn 是 Facebook, Google, Exponent 和 Tilde 开发的一款新的 JavaScript 包管理工具。目的是解决这些团队使用 npm 面临的一些问题

为什么使用 yarn

安装速度更快

  • 在 npm 中执行包安装任是按包的顺序一个个执行，所以必须等待上一个包被完整安装才会进入下一个
  • Yarn 则并行的执行这些任务，提高了性能。

更清晰的输出

  • 比 npm 更直观的信息提示

CLI 命令

yarn global

  • npm 添加 -g 或 --global 可以进行全局安装，Yarn 使用的是 global 前缀。
  • global 前缀只能用于 yarn add, yarn bin, yarn ls 和 yarn remove，除yarn add外，这些命令都和 npm 等效

NVM, NPM的版本管理工具

发表于 2018-01-18 | 分类于 tools

卸载已经安装的 node/npm

  • node 安装目录
    • node 目录
      • /usr/local/bin/node
    • npm 目录
      • 在全局 node_modules 目录中，具体路径为 /usr/local/lib/node_modules/npm
  • 卸载当前 node
    • npm ls -g --depth=0
      • 查看已经安装在全局的模块，方便重新安装
    • sudo rm -rf /usr/local/lib/node_modules
      • 删除全局 node_modules
    • sudo rm /usr/local/bin/node
      • 删除 node
    • cd /usr/local/bin && ls -l | grep "../lib/node_modules/" | awk '{print $9}'| xargs rm
      • 删除全局 node 模块注册的软链
